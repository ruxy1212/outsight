# Outsight

A small React + Vite app that generates a useless “cosmic judgment” by fetching random country, quote, and fact data.

## Run locally

1. Clone:

```bash
git clone <your-repo-url>
cd outsight
```

2. Install dependencies:

```bash
pnpm install
```

3. Start dev server:

```bash
pnpm dev
```

Then open the local URL shown in your terminal (usually `http://localhost:5173`).
